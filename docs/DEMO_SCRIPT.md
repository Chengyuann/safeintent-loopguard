# 90-second Demo Script

## Recording Goal

Show SafeIntent LoopGuard as an OKX.AI ASP that protects the step before an agent pays, signs, approves, or loops.

Keep the recording under 90 seconds.

## Shot Plan

### 0-8s - Hero

Show the one-screen homepage.

Voiceover:

> SafeIntent LoopGuard is a pre-execution firewall for OKX.AI agents. It starts before the wallet popup, when social persuasion becomes an agent task.

### 8-18s - Four Modules

Hover or point across the four homepage modules:

- Social Context
- Mandate Compiler
- Loop Guard
- Intent Receipt

Voiceover:

> It turns social voice or community context into policy, checks the agent plan, and returns an explainable receipt.

### 18-36s - Social Context

Click `Social Context`.

Use the fake urgent airdrop scenario.

Voiceover:

> In this scenario, a Discord AMA claims users must upgrade their wallet before midnight. SafeIntent treats this as risk-bearing social context, not just plain text.

### 36-54s - Mandate Compiler

Close, click `Mandate Compiler`, or switch in the overlay flow.

Voiceover:

> The user's mandate says: main wallet only for checking, no unknown approvals, no unlimited approval, paid budget under five USDT. SafeIntent compiles that into policy.

### 54-72s - Loop Guard

Click `Loop Guard`.

Voiceover:

> The agent plan starts with a paid eligibility check, then drifts into an unlimited approval from the main wallet. That violates the user's read-only intent and wallet policy.

### 72-88s - Receipt

Click `Intent Receipt`.

Voiceover:

> SafeIntent returns BLOCK with the exact conflicts and a safer rewrite. Other OKX.AI agents can call this ASP before MCP calls, x402-style payments, approvals, signatures, and repeated paid loops.

## Optional Second Scenario Cutaway

If time allows, switch to `Paid MCP loop` and show that SafeIntent returns `ASK_MORE` before the agent keeps spending through repeated paid tool calls.

## Must Show

- Homepage module layout.
- One module click opening the detail overlay.
- At least one `BLOCK` decision.
- Conflict evidence.
- Safer rewrite or receipt.

## Do Not Show

- Private keys or real wallet execution.
- Any claim that SafeIntent provides financial advice.
- A demo longer than 90 seconds.
