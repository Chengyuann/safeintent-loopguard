## Visual Rules

- Use the real product UI as the primary visual.
- Keep cursor movement slow and intentional.
- Use hard cuts on musical scene-change hits; use short dissolves only for intro and close.
- Avoid decorative stock footage or unrelated crypto imagery.
- Keep captions inside a bottom safe area and away from interactive controls.
- Use cyan for product/proof labels, amber-red for BLOCK evidence, and white for narration captions.
