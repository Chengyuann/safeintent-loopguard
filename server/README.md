# SafeIntent API Server

Run the local API server:

```bash
npm run dev:server
```

Default URL:

```text
http://127.0.0.1:8787
```

Endpoints:

- `GET /api/health`
- `GET /api/scenarios`
- `GET /api/asp/manifest`
- `POST /api/intake/social-risk`
- `POST /api/mandate/compile`
- `POST /api/guard/check`
- `POST /api/receipt/generate`
